# Tree And Postfix Expression

Question: (a + b + c + d) / (e –f + g)

Answer: ab+c+d+ef-g+/

Question: ((a + b)/c*(d + e))/f

Answer: ab+c/de+*f/

Question: (a * (b + c -d) * e ) / (f + g)

Answer: abc+d-*e*fg+

Question: (a * (b + c) / d + (e –f)) * (g –h)

Answer: abc+*d/ef-+gh-*

Question: (a + b)/c + (d + e)/f + (g + h)/i

Answer: av+c/de+f/+gh+i/+
